# InsightIQ - Enterprise Business Intelligence Platform

## Overview

InsightIQ is an Enterprise Data Profiling and Visualization Platform built using FastAPI and Pandas.

## Features

- CSV Upload
- Dataset Summary
- Statistical Summary
- Missing Value Analysis
- Duplicate Detection
- Data Type Detection
- Correlation Analysis
- Outlier Detection
- Business Insight Generation
- Histogram Visualization
- Report Export

## Technology Stack

- Python
- FastAPI
- Pandas
- NumPy
- Matplotlib
- Uvicorn
